TelegramEvent

rawEvent​

Underlying raw event from Telegram.

Example:

event.rawEvent;
// {
//   message: {
//     messageId: 666,
//     from: {
//       id: 427770117,
//       isBot: false,
//       firstName: 'first',
//       lastName: 'last',
//       languageCode: 'en',
//     },
//     chat: {
//       id: 427770117,
//       first_name: 'first',
//       last_name: 'last',
//       type: 'private',
//     },
//     date: 1499402829,
//     text: 'text',
//   },
// }

isMessage​

Determine if the event is a message event.

Example:

event.isMessage; // true

The message object from Telegram raw event.

Example:
